# Getting Started with Kronos

Welcome to Kronos! This guide will help you get up and running in minutes.

<Callout type="info">
  Kronos is currently in **public beta** (v0.4.0). We'd love your feedback!
</Callout>

## Installation

<Steps>
<Step>

### Install Dependencies

<Tabs items={['macOS', 'Ubuntu/Debian', 'Arch Linux', 'Windows']}>
<Tab value="macOS">
```bash
# Install Xcode Command Line Tools (includes Clang)
xcode-select --install

# Install gperf via Homebrew
brew install gperf
```
</Tab>
<Tab value="Ubuntu/Debian">
```bash
sudo apt-get update
sudo apt-get install build-essential gperf
```
</Tab>
<Tab value="Arch Linux">
```bash
sudo pacman -S base-devel gperf
```
</Tab>
<Tab value="Windows">
**Option 1: WSL (Recommended)**

Install [WSL](https://learn.microsoft.com/en-us/windows/wsl/install), then follow Ubuntu/Debian instructions:
```bash
wsl --install
# After restart, in WSL terminal:
sudo apt-get update
sudo apt-get install build-essential gperf
```

**Option 2: MSYS2**

1. Install [MSYS2](https://www.msys2.org/)
2. Open MSYS2 UCRT64 terminal:
```bash
pacman -S mingw-w64-ucrt-x86_64-gcc mingw-w64-ucrt-x86_64-gperf make
```
</Tab>
</Tabs>

</Step>
<Step>

### Clone & Build

```bash
git clone https://github.com/nedanwr/kronos.git
cd kronos
make
```

<Callout type="warn">
  The build process requires **GCC** or **Clang** and **gperf** for keyword hashing.
</Callout>

<Callout type="info">
  **Windows users:** Run these commands in WSL or MSYS2 terminal. The output will be `kronos` (WSL) or `kronos.exe` (MSYS2).
</Callout>

</Step>
<Step>

### Verify Installation

```bash
./kronos --version
```

You should see something like:
```
Kronos 0.4.0
```

<Callout>
  **Windows (MSYS2):** Use `./kronos.exe --version` or add to PATH.
</Callout>

</Step>
</Steps>

## Your First Program

Create a file called `hello.kr`:

```kronos title="hello.kr"
print "Hello, World!"
```

Run it:

```bash
./kronos hello.kr
```

```ansi title="Output"
Hello, World!
```

## Interactive REPL

Kronos includes a powerful REPL for experimenting with code:

```bash
./kronos
```

```kronos title="REPL Session"
> 10 plus 20
30
> set name to "Kronos"
> print f"Hello, {name}!"
Hello, Kronos!
```

<Callout>
  **Pro tip:** The REPL supports arrow keys for history, tab completion, and multi-line input!
</Callout>

## Command-Line Options

| Option | Description |
|--------|-------------|
| `kronos script.kr` | Execute a Kronos file |
| `kronos -e "code"` | Execute code directly |
| `kronos --version` | Show version |
| `kronos --help` | Show all options |

**Examples:**
```bash
# Execute code directly
./kronos -e "print 42"

# Chain multiple expressions
./kronos -e "set x to 10" -e "print x times 2"
```
